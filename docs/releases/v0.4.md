# v0.4 Release Notes

## v0.4.x Maintenance Scope

v0.4.x is a non-semantic maintenance line.

Allowed changes:
- migration hygiene
- output layout stabilization

Forbidden changes:
- core semantics (`base_score`, `boosted_score`, `delta_boost`, `event_strength`, alias SSOT)
- acceptance behavior and deterministic core hash output

This line exists to keep v0.4 clean, handoff-ready, and stable as the baseline for v0.5.
