# v0.7 Release Notes

## v0.7 PR-8 Scope (Persistence Track)

This PR adds observation-only persistence outputs and dashboard visualization.

- Adds `persistence_v1` (tag-level persistence / ERS).
- Adds `event_kernel_v1` (domain-level explanatory kernel).
- Adds deterministic derived outputs under `output/.../derived/`.
- Adds a dashboard panel:
  - `Persistence (Observation)`
  - Includes fixed note: `Observation-only. Not used in scoring or gating.`

## Non-Interference Contract

PR-8 does **not** modify:

- `strength` semantics
- `boosted_push` semantics
- gate/event level semantics
- pipeline scoring path

Persistence outputs are render/derived-only and are not used by gate/scoring logic.
